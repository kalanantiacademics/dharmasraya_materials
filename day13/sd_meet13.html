Dari Mana Dapat Aset?

1. Scratch Library

Paling mudah! Klik tombol "Choose a Sprite" atau "Choose a Backdrop". Ada ribuan gambar siap pakai.

2. Internet (Google)

Cari gambar di Google. Tips: Tambahkan kata "Cartoon" atau "PNG Transparent" agar gambarnya bagus.

3. Gambar Sendiri

Gunakan fitur "Paint" di Scratch untuk menggambar karakter unikmu sendiri!

4. Foto Kamera

Kamu bisa memfoto gambarmu di kertas lalu upload ke komputer.

Cara Upload Gambar

  1. Download gambar dari Google (Klik Kanan -> Save Image As).
  2. (Opsional) Buka remove.bg jika ingin menghapus latar belakang putih.
  3. Buka Scratch, arahkan mouse ke ikon "Kucing" (Choose a Sprite).
  4. Klik tombol paling atas: Upload Sprite (ikon panah ke atas).
  5. Pilih file gambarmu, lalu klik Open.
