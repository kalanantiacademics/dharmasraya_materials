Proyek Game: Zombie Survival!

Selama beberapa pertemuan ke depan, kita akan membuat game seru ini! Player harus bertahan dari serangan para zombie.

Langkah 1: Siapkan Aset Game!

Ayo kita kumpulkan semua gambar yang dibutuhkan. Unggah semua gambar ini sebagai Sprite dan Backdrop baru di proyek Scratch-mu.

Karakter (Sprites)

  • Player (Karakter Utama)
  • Zombie Biasa
  • Bos Zombie
  • Nyawa (misal: gambar hati)

Latar (Backdrops)

  • Latar Belakang Awal
  • Latar Belakang Game
  • Latar Belakang Game Over

Tips: Kamu bisa mencari gambar di internet dengan kata kunci "character top down png", "zombie top down png", atau "game background".

Memahami Peta: Sumbu X dan Y

Panggung Scratch itu seperti peta. Posisi setiap sprite ditentukan oleh koordinat X (horizontal) dan Y (vertikal).

Sumbu X

Mengatur posisi kiri (-) dan kanan (+). Titik tengah adalah X=0.

Sumbu Y

Mengatur posisi bawah (-) dan atas (+). Titik tengah adalah Y=0.

Kategori Events: Si Pemicu Aksi!

Blok kuning ini adalah "kejadian" yang memulai sebuah kode. Tanpa blok ini, kode tidak akan pernah berjalan.

when green flag clicked

Ini seperti tombol 'MULAI' pada game. Kode di bawahnya akan jalan saat bendera hijau ditekan.

when ... key pressed

Memulai kode ketika kita menekan tombol tertentu di keyboard (misal: spasi, panah atas, huruf W).

Kategori Motion: Si Penggerak Aksi!

Blok biru ini bertanggung jawab untuk semua jenis gerakan dan posisi sprite di panggung.

move ... steps

Menggerakkan sprite maju sesuai arahnya saat ini. Seperti menyuruh robot "Maju!".

change x by ...

Menggeser posisi sprite ke kiri (nilai negatif) atau ke kanan (nilai positif).

change y by ...

Menggeser posisi sprite ke bawah (nilai negatif) atau ke atas (nilai positif).

point towards ...

Membuat sprite menghadap ke arah mouse atau sprite lain.

Tantangan: Gerakan 4 Arah!

Sekarang giliranmu! Bisakah kamu membuat karakter bergerak ke atas dan ke bawah? Coba juga ganti kontrolnya menggunakan tombol W, A, S, D!

Petunjuk:

  • Untuk bergerak atas/bawah, lebih mudah menggunakan blok .
  • Gunakan nilai positif (misal: 10) untuk ke atas.
  • Gunakan nilai negatif (misal: -10) untuk ke bawah.
  • Kamu bisa mengganti event menjadi tombol lain seperti W, A, S, atau D.

Langkah 3: Selalu Waspada!

Agar karakter kita bisa menembak ke arah yang benar, ia harus selalu menghadap ke arah mouse. Gunakan kombinasi blok ini!

Artinya: "Ketika bendera hijau diklik, ulangi selamanya, buat karakter menghadap ke arah penunjuk mouse."
