Konsep: Interaksi Fisik (Drag & Touch)

Daripada "menyimpan" barang di memori, kita akan buat barangnya bisa digeser (Drag) dan kita akan cek apakah barang itu "menyentuh" (Touching) benda lain.

Blok Sensor Kunci #1:

set drag mode [draggable]
Blok ini ada di kategori Sensing (Sensor). Fungsinya agar sprite bisa di-klik dan digeser-geser oleh pemain!

Konsep: Interaksi Fisik (Drag & Touch)

Blok Sensor Kunci #2:

touching [ ... ] ?
Blok Sensing berbentuk hexagonal ini adalah sebuah 'Kondisi'. Dia akan bernilai 'Benar' jika sprite ini menyentuh sprite lain yang kita pilih.

Misi 1: Menggeser `Karpet`

Ini adalah puzzle pertama. Saat `Karpet` di-klik, dia harus bergeser.

Kode di sprite `Karpet`:

  1. Gunakan when this sprite clicked.
  2. Tambahkan glide 1 secs to x: [posisi baru] y: [posisi baru].
  3. Pindahkan karpet di stage ke posisi baru, lalu catat angka X dan Y nya.
Fungsi Blok:
when this sprite clicked (Events): Kartu 'event' ini akan memulai kode HANYA saat sprite ini di-klik.

glide 1 secs to... (Motion): Blok 'motion' (gerakan) ini akan menggerakkan sprite secara perlahan ke posisi baru, bukan instan (teleport).

PENTING: Pastikan kode "Reset" dari Pertemuan 8 (when backdrop switches to Room 1 -> go to [posisi awal]) masih ada!

Misi 2: Clue di `Frame Foto`

Saat `Frame Foto` di-klik, kita akan beri petunjuk kepada pemain.

Kode di sprite `Frame Foto`:

  1. Gunakan when this sprite clicked.
  2. Tambahkan say [ketik petunjukmu di sini...] for 3 secs.
  3. Contoh petunjuk: "Alat itu sepertinya bisa memecahkan sesuatu..." (Hint untuk Palu & TV)
Fungsi Blok:
when this sprite clicked (Events): Sama seperti karpet, kode ini berjalan saat frame di-klik.

say ... for ... secs (Looks): Blok 'looks' (tampilan) ini akan memunculkan gelembung teks di atas sprite selama beberapa detik.

Misi 3: Membuat `Palu` Bisa Di-Drag

Kita akan buat `Palu` bisa digeser-geser oleh pemain.

Kode di sprite `Palu`:

1. Kode "Reset" (PENTING):
Saat when backdrop switches to Room 1:

  • show (Palu muncul)
  • go to x: [posisi awal] y: [posisi awal]
  • set drag mode [draggable] (Biar bisa di-drag!)

2. Kode "Sembunyi":
Saat when backdrop switches to Room 2 (dan 3):

  • hide (Sembunyi di ruangan lain)

Fungsi Blok:
go to x: y: (Motion): Blok 'reset' yang penting. Memastikan Palu kembali ke tempatnya setiap masuk Room 1.

set drag mode [draggable] (Sensing): BLOK BARU! Ini adalah blok 'sensor' yang mengizinkan pemain meng-klik dan menahan sprite ini untuk digeser-geser.

Misi 4: Memecahkan `TV`

Saat `TV` di-klik, kita cek: "Apakah `Palu` sedang menyentuh TV?"

Kode di sprite `TV`:

  1. Gunakan when this sprite clicked.
  2. Tambahkan blok if ... then ... else.
  3. Di dalam if, masukkan kondisi touching [Palu] ?.
  4. Jika YA (Menyentuh Palu):
    • switch costume to [tv-pecah]
    • broadcast [tvPecah] (Untuk memunculkan kunci!)
  5. Jika TIDAK (else):
    • say "Butuh alat untuk ini!" for 2 secs
Fungsi Blok:
if...then...else (Control): Blok 'kontrol' ini adalah 'persimpangan'. JIKA kondisinya benar, jalankan 'then'. JIKA salah, jalankan 'else'.

touching [Palu]? (Sensing): Kondisi 'sensor' ini akan bernilai 'benar' JIKA sprite TV sedang bersentuhan dengan sprite Palu.

broadcast [tvPecah] (Events): Mengirim 'sinyal' rahasia ke semua sprite lain (terutama ke Kunci).

Reset: Pastikan kode reset switch costume to [tv-normal] dari Meet 8 still ada (di when backdrop switches to Room 1)!

Misi 5: `Kunci` Muncul & Di-Drag

`Kunci` akan merespon `broadcast` dari TV, lalu bisa kita geser juga.

Kode di sprite `Kunci`:

1. Kode "Reset" (PENTING):
Saat when backdrop switches to Room 1 (dan 2, 3):

  • hide (Kunci harus selalu sembunyi di awal)

2. Kode "Muncul":
Saat when I receive [tvPecah]:

  • show
  • go to x: [posisi TV] y: [posisi TV]
  • set drag mode [draggable]

Fungsi Blok:
when I receive [tvPecah] (Events): Event ini 'mendengar' sinyal dari TV. Saat TV pecah, kode ini berjalan.

show / hide (Looks): Memunculkan atau menyembunyikan kunci.

set drag mode [draggable] (Sensing): Agar kuncinya juga bisa kita ambil dan geser!

Misi 6: Membuka `Pintu`!

Misi Terakhir! Saat `Pintu` di-klik, kita cek: "Apakah `Kunci` menyentuh Pintu?"

Kode di sprite `Pintu`:

  1. Gunakan when this sprite clicked.
  2. Tambahkan blok if ... then ... else.
  3. Di dalam if, masukkan kondisi touching [Kunci] ?.
  4. Jika YA (Menyentuh Kunci):
    • broadcast [bukaPintu] (Sinyal dari Meet 7!)
  5. Jika TIDAK (else):
    • say "Terkunci!" for 2 secs
Fungsi Blok:
if (Sensing/Control): Logika yang sama dengan TV! Pintu akan cek, 'Apakah Kunci menyentuhku?'

broadcast [bukaPintu] (Events): Jika ya, kirim sinyal 'bukaPintu' yang sudah kita buat di Meet 7 untuk pindah ruangan.

say "Terkunci!" (Looks): Pesan jika kita klik pintu tanpa membawa kunci.
