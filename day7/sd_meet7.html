Blok Fokus: Variabel (Oranye Tua)

Ini adalah 3 blok utama kita hari ini. Ayo kita coba!

Tantangan Mini: Buat Variabel 'Skor'

Ayo kita buat "Papan Skor" kita!

  1. Klik kategori Variables (Oranye Tua).
  2. Klik tombol "Make a Variable".
  3. Beri nama variabel baru kita: Skor
  4. Pastikan memilih "For all sprites".
  5. Klik OK.

Lihat! Papan Skor kita sudah muncul di layar!

Langkah 1: Atur Skor di Stage

Tempat terbaik untuk mengatur ulang game adalah di Stage/Backdrop. Ayo klik sprite Stage (di sebelah kanan).

Susun blok ini:

  • When Green Flag Clicked
  • set [Skor] to 0

Blok set artinya "ATUR JADI". Ini seperti menghapus papan tulis dan menulis angka 0.

Blok 'Change' (Ubah)

Lalu, bagaimana cara menambah skor? Kita pakai blok change (Ubah).

change [Skor] by 1 artinya:
"Ambil angka yang ada di Skor, lalu tambahkan 1."

Blok ini akan kita pakai nanti kalau kita berhasil membuang sampah.
