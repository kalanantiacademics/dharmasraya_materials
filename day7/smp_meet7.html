Misi 1: Siapkan Ruangan (Backdrop)

Escape Room yang seru punya banyak area! Ayo kita siapkan "Scene" atau "Ruangan" kita.

  1. Pergi ke Stage/Backdrop (pojok kanan bawah).
  2. Klik tab Backdrops (kiri atas).
  3. Ayo buat 3 Backdrop:
    • "Halaman Depan" (Beri tombol Start di sini)
    • "Ruangan 1" (Ruangan utama kita)
    • "Ruangan 2" (Ruangan sebelah)

Misi 2: Buat Tombol Start

Ayo buat Sprite baru (gambar kotak/lingkaran, beri tulisan "START"). Beri nama: Tombol Start.

[Gambar blok when this sprite clicked dan switch backdrop to...]

Tantangan Mini:

Bagaimana cara membuat tombol ini berfungsi? Kita ingin:
SAAT DI-KLIK -> PINDAH KE RUANGAN 1.

Solusi: Pasang kode ini di sprite Tombol Start.

Masalah: Tombolnya Masih Ada!

Saat kita pindah ke Ruangan 1... LHO??? Kenapa Tombol Start-nya MASIH ADA?

Problem

[Gambar Ruangan 1 dengan Tombol Start besar di tengah, beri tanda tanya]

Sprite tidak tahu kapan harus sembunyi.

Konsep Utama: Manajemen Scene

Kita harus membuat sprite kita "PINTAR". Sprite harus tahu:

  • "Oh, aku di Halaman Depan, aku harus MUNCUL."
  • "Oh, aku pindah ke Ruangan 1, aku harus SEMBUNYI."

Blok Fokus: Blok Ajaib!

Blok ini memberi tahu SEMUA sprite setiap kali kita pindah ruangan. Kita bisa pakai ini untuk memberi aturan show atau hide.

Events (Kuning)

Artinya: "Ketika latar belakang berganti ke..."

Misi 3: Membuat Tombol Start "Pintar"

Kembali ke sprite Tombol Start. Kita tambahkan 3 aturan ini:

Penjelasan Kode:

  • when green flag: Memastikan game SELALU mulai dari Halaman Depan & tombolnya terlihat.
  • when backdrop... [Halaman Depan]: Jika kita KEMBALI, tombolnya muncul lagi.
  • when backdrop... [Ruangan 1]: SAAT masuk Ruangan 1, tombolnya langsung sembunyi!

Misi 4: Siapkan Objek Ruangan 1

Sekarang, ayo buat 2 sprite baru: "Pintu" dan "Rak Buku". Benda ini HANYA boleh muncul di Ruangan 1.

Solusi Kode (di sprite Pintu):

Sprite "Pintu" dan "Rak Buku" harus "Pintar".

  • ... [Halaman Depan] -> hide
  • ... [Ruangan 1] -> show
  • ... [Ruangan 2] -> hide

Lakukan hal yang sama untuk sprite "Rak Buku"!

Misi 5: Pintu yang Bereaksi

Pintu di Ruangan 1 harus bisa terbuka. Tapi tidak dengan di-klik, melainkan saat menerima "sinyal rahasia".

Konsep: Ganti Kostum!

  1. Di sprite Pintu, buat 2 kostum: "tertutup" dan "terbuka".
  2. Saat game mulai di Ruangan 1, pastikan pintu memakai kostum "tertutup".
  3. Saat menerima sinyal (broadcast) "buka pintu", ganti kostumnya menjadi "terbuka".
  4. Tambahkan kode di samping.

when I receive...: Blok ini akan menunggu "sinyal radio" sebelum menjalankan perintah.
