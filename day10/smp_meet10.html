Judul Modal

Kita akan menggunakan broadcast untuk memberi sinyal kapan game harus dimulai. Semua objek game akan sembunyi dulu.

  1. Buat Backdrop baru untuk Layar Judul. Buat juga Sprite baru untuk tombol "Mulai".
  2. Di Stage (Backdrop), tambahkan when green flag clicked -> switch backdrop to [Layar Judul].
  3. Di Sprite Tombol Mulai, tambahkan when green flag clicked -> show. Lalu, when this sprite clicked -> broadcast [Mulai Game] -> hide.
  4. Di semua sprite game-mu (kunci, pintu, dll), tambahkan when green flag clicked -> hide.
  5. Lalu, di sprite game itu juga, tambahkan when I receive [Mulai Game] -> show. (Dan di Stage: when I receive [Mulai Game] -> switch backdrop to [Ruangan 1]).

Kita bisa memunculkan petunjuk saat objek lain diklik. Misalnya, mengklik lukisan akan memunculkan kertas petunjuk.

  1. Buat Sprite baru untuk "Kertas Petunjuk". Sembunyikan di suatu tempat (misal, di belakang lukisan).
  2. Di Sprite Kertas Petunjuk, tambahkan when green flag clicked -> hide.
  3. Buat Sprite "Lukisan" (atau objek lain untuk diklik).
  4. Di Sprite Lukisan, tambahkan when this sprite clicked -> broadcast [Lihat Petunjuk].
  5. Kembali ke Sprite Kertas Petunjuk, tambahkan when I receive [Lihat Petunjuk] -> show.
  6. (Opsional) Buat petunjuknya hilang lagi setelah beberapa detik, atau saat diklik.

Pindah ruangan berarti kita mengganti Backdrop dan menyembunyikan/menampilkan objek yang sesuai. Kita butuh broadcast lagi!

  1. Buat Backdrop baru, beri nama "Ruangan 2".
  2. Di Sprite Pintu (di Ruangan 1), ubah kodenya. when this sprite clicked -> if <[Punya Kunci] = 1> then -> broadcast [Pindah Ruangan 2] -> hide.
  3. Di Stage (Backdrop), tambahkan when I receive [Pindah Ruangan 2] -> switch backdrop to [Ruangan 2].
  4. Di semua sprite Ruangan 1 (kunci, lukisan, dll), tambahkan when I receive [Pindah Ruangan 2] -> hide.
  5. Buat sprite baru untuk Ruangan 2 (misal: brankas). Di sprite itu, tambahkan when green flag clicked -> hide. Lalu when I receive [Pindah Ruangan 2] -> show.

Ini cara keren untuk membuat teka-teki! Kita akan pakai blok ask dari kategori Sensing.

  1. Buat Sprite baru untuk "Brankas".
  2. Di Sprite Brankas, tambahkan when this sprite clicked -> ask [Masukkan kode (4 angka):] and wait.
  3. Tepat di bawahnya, tambahkan blok if <...> then ... else ... dari Control.
  4. Isi kondisinya dengan blok (...) = (...) dari Operator. Masukkan [answer] (dari Sensing) di slot kiri, dan ketik kodemu (misal: 1234) di slot kanan.
  5. Di dalam if (jika benar): say [Kode Benar!], lalu broadcast [Brankas Terbuka], dan switch costume to [Brankas Terbuka].
  6. Di dalam else (jika salah): say [Kode Salah! Coba lagi.].

Bagaimana cara game "mengingat" barang yang sudah kita ambil? Pakai Variabel!

  1. Pergi ke Variables, klik "Make a Variable". Beri nama Punya Kunci (pilih "For all sprites").
  2. Di Stage, tambahkan when green flag clicked -> set [Punya Kunci] to 0. (Artinya, di awal game, kita belum punya kunci).
  3. Di Sprite Kunci, tambahkan when this sprite clicked -> set [Punya Kunci] to 1 -> hide. (Sekarang game "ingat" kita punya kunci, lalu kuncinya hilang).
  4. Di Sprite Pintu, ubah logikamu. Ganti if <touching [Kunci]?> menjadi if <(Punya Kunci) = 1>.
  5. Sekarang, kamu hanya perlu mengklik pintu (tanpa harus menyeret kunci) dan jika variabel Punya Kunci adalah 1, pintu akan terbuka!
