Review: Helikopter Penyelamat!

Di pertemuan sebelumnya, kita sudah berhasil membuat helikopter bergerak mengikuti mouse. Hebat!

Langkah 1: Pilih Sprite Api!

Buka kembali proyek game pemadam kebakaran yang sudah kita buat kemarin. Pastikan kamu sudah login ya!

  1. Buka proyek dari pertemuan sebelumnya.
  2. Di area daftar sprite (kanan bawah), cari dan klik sprite Api.
  3. Area skrip sekarang siap untuk kita beri kode!

Misi Kita: Membuat Api Berkobar!

Hari ini, kita akan membuat api yang tadinya diam menjadi berkobar seperti ini menggunakan beberapa kostum.

Konsep Baru: Muncul & Sembunyi!

Di kategori Looks (ungu), ada dua blok penting: show (tampilkan) dan hide (sembunyikan).

show

Membuat sprite terlihat di panggung.

hide

Membuat sprite menghilang dari panggung.

Langkah 2: Sembunyi & Tentukan Posisi

Saat game dimulai, kita mau apinya sembunyi dulu. Lalu, kita tentukan di mana api pertama akan muncul.

Susun kode ini di sprite Api:

  • Ambil .
  • Pasang di bawahnya.
  • Pasang .

Tips: Geser sprite api ke posisi yang kamu mau, lalu lihat angka di blok `go to x: y:` akan berubah otomatis!

Langkah 3: Munculkan Apinya!

Setelah beberapa detik, misalnya 2 detik, kita akan munculkan apinya.

Tambahkan blok wait 2 seconds dari kategori Control (oranye), lalu tambahkan blok show dari Looks (ungu).

Langkah 4: Buat Apinya Berkobar!

Agar apinya 'berkobar', kita suruh dia ganti kostum terus-menerus.

Masukkan blok forever, lalu di dalamnya ada next costume dan wait 0.5 seconds. Blok `wait` penting agar animasinya tidak terlalu cepat!
